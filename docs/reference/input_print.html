5. 基本输入和输出¶

AIL提供了最基本的对 stdout 和 stdin 写入和读取的操作方式: print 和 input 。input 和 print 在AIL中都已 关键字 的形式存在。

5.1. print语句¶

print语句是最简单的向输出流写入数据的方式。其语法如下：

print expr [';' expr]*

expr 可以是任意表达式。多个表达式用 ; 隔开。

5.2. input语句¶

input语句是最简单的向输入流读取数据的方式。其语法如下：

input expr [';' NAME [',' NAME]*]*

expr 可以是任意表达式，作为提示 (prompt) 使用。 NAME 作为存储输入的变量名，可以有多个 NAME ，用 ; 隔开。当input语句成功执行完毕后，会在 当前名称空间 中储存输入内容到该作为该变量名的变量。如果是有多个变量名，则会对输入的内容使用 空格 分割，再分别按顺序存入各个变量中。如果又提供了多个变量名，却按照分割的数量和提供的变量名数量 不相等 ，则抛出 ValueError 异常。

5.3. 题外话：更为灵活的输入输出¶

若想进行更灵活的控制台输入输出，可以使用内置的 console 对象。
