AIL 2.1 Leopard¶

2021年6月18日

2.1 Leopard版本相比1.2 DragonLi版本，提高了稳定性，切换了默认运行时。

主要更新¶

  1. 运行时更换为 Python 运行时，焕然一新的运行速度！ AIL运行时仍然保留，只不过默认处于关闭状态

  2. 新的 for 语法！

    • for 语句将有 3 种形式: forever, condition-only, classic

    • forever: for {...}

    • condition-only: for expr {...}

    • classic: for [init] ; [condition] ; [update] {...}

    • 仍然可以使用旧的 for 语法

  3. 类型注释语法，便于静态类型检查器检查代码的正确性

  4. 元组的加入！您可以使用 , 分隔数据来声明元组！ 一般来说建议使用 (…) 声明元组

  5. 数据解包操作！
